### Description

This script automates leveling up by operating in cycles. You simply enter the target level you want to reach, and the script calculates the exact number of cycles required. It then executes the necessary requests in pairs (one start request followed by one complete request) until the target level is achieved.

---

### Logic

**Target:** Level 5  
**Calculation:** 14 cycles per level = **70 cycles total** (70 start + 70 complete).  
**Estimated Time:** ~1 minute 10 seconds.  

---

### Installation

**1. Manager** Install Tampermonkey or Violentmonkey.  

**2. Script** Click on the `.user.js` file in this repository and select **Raw**.  

**3. Confirm** Confirm the installation in your browser extension.
